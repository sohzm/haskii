haskii

pretty self-explanatory, really. produces ascii art from images. and it's written in haskell. and it's called haskii. get it?

<br>

```
~/haskii $ cabal install --only-dependencies
~/haskii $ cabal build
~/haskii $ cabal install --installdir=$HOME/.local/bin
~/haskii $ haskii --input extra/elon.jpg  --scale 0.03
```
